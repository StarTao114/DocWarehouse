* details
  * 预计轮询速度为8个节点/2s，一分钟内轮询完200个节点


* 各单元LED效果及其对应特点
  * 独立电台
    * 开始轮询第一组时，LED0亮，其余情况灭
    * 正常运行时，LED1闪烁
  * 普通电台
    * 周期变化时，LED0翻转状态
    * 成功接收时亮，未接收或接收错误时灭
  * 节点
    * 跳频后开始发送时，LED0亮；跳回基础频段开始监听时，LED0灭
    * 成功接收时亮，未接收或接收错误时灭
